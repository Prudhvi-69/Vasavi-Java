next() : used to take a single String input
nextLine() : used to take a multi word String input
nextInt() : used to take integer input 
nextDouble() : used to take float/double input 
nextByte() : used to take a Byte of data
nextShort() : used to take Short integer data

Datatypes in Java 

Byte		1 byte of data 			- numeric values
Short    2 bytes of data 		   - numeric values more than byte
Int 			4 bytes of data 			- numeric values more than short
Long 		8 bytes of data 			- numeric values more than int
Float 		4 bytes of data 			- decimal values, upto precision 6
Double 8 bytes of data 			- decimal values, upto 12 
Char 		1 byte of data 			- Single character data
Boolean 	Depends on JVM - Takes a true or false value 


Automatic type conversion : 

Byte -> Char -> Short -> Int -> Long -> Float -> Double 

Manual Type conversion : 

Double -> Float -> Long -> Int -> Short -> Char -> Byte
